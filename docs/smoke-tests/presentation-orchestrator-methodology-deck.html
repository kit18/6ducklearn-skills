Core finding

The skill is useful because it routes before it writes

Generic deck prompts jump straight to slide titles. This skill first names the communication job, then chooses the method that should shape the deck.

Must decide?
Pyramid Principle
Answer-first recommendation and grouped reasons.
Problem unclear?
Hypothesis-led solving
Issue tree, tests, options, and evidence.
Data-heavy?
Data storytelling
One implication per chart and clear annotation.

Best design finding

The best default is an executive consulting/data-story deck

Weak default Dense report pasted into slides

Topic titles, many bullets, unclear ask, and charts without implications.

Skill default Answer-first, evidence-backed slides

Action titles, one claim per slide, one primary visual, and a clear decision ask.

Evidence: presentation-design-patterns.md5
Slide rule

Every slide should do one job and carry one claim

Action-title claim goes here

Footer: source, caveat, or implication.

Title says the takeaway
Body proves that takeaway
Visual is chosen by message
Speaker note names the transition
