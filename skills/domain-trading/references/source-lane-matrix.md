# Source lane matrix

## Timing inputs
- X or Twitter signals
- news and RSS
- Polymarket changes
- short-term OHLCV shifts
- Reddit, web, or Instagram buzz
- GitHub activity when it maps to release timing

## Value inputs
- revenue growth
- margin trends
- product strength
- moat evidence
- stock screener fundamentals
- saved research and thesis history

## Risk inputs
- volatility
- balance sheet weakness
- event concentration
- liquidity
- source conflict
- fraud or manipulation risk

Rule:
- attention is not value
- narrative is not margin of safety
