# Trading research output template

Use this as a reference template, not a universal standard.
Adapt it to the task.

## Summary verdict
- verdict
- confidence
- horizon

## Value view
- fair value range
- current price context
- margin of safety

## Timing view
- current catalysts
- signal quality
- entry zone
- do-not-chase level

## Risk view
- bear case
- invalidation level
- what would make this a terrible investment?

## Evidence
- value evidence
- timing evidence
- conflicting evidence

## Follow-up
- what to monitor next
- what changes the conclusion
