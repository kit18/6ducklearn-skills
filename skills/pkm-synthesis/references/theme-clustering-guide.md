# Theme Clustering Guide

Cluster around meaning, not just repeated nouns.

## Good clusters
- Repeated concern about timing.
- Repeated belief about moat strength.
- Recurring pattern of overestimating upside.
- Consistent note that a user values simplicity over scale.

## Weak clusters
- Superficial keyword overlap.
- One-off comments with no recurrence.
- Emotional tone without decision relevance.

## Rule
- Prefer fewer high-signal themes over many weak categories.
