# Methodology Catalog

Use this catalog when the user asks what methods exist, wants source anchors, or needs a buying/learning path. The skill routes across three layers: framework library, consultant problem-solving, and strategy-to-deck expression.

## Layer 1: Framework Library

These sources help pick the right analytical model for a slide or diagnosis.

| Source | Methods captured | Best use in a deck | Source anchor |
|---|---|---|---|
| Key Management Models | SWOT, Porter Five Forces, BCG, Ansoff, 7S, value chain, Business Model Canvas, and many other management models | Use as the framework encyclopedia when the user asks "which model fits this business question?" | Google Books: https://books.google.com/books/about/Key_Management_Models.html?id=HrU8KaQnNJYC |
| HBR's 10 Must Reads on Strategy | Porter strategy fundamentals, competitive forces, strategic positioning | Use when the deck needs strategy substance behind a model, not only a diagram | HBR/Amazon listing: https://www.amazon.com/HBRs-10-Must-Reads-Strategy/dp/1422157989 |
| Understanding Michael Porter, Joan Magretta | Five Forces, competitive advantage, tradeoffs, value chain, differentiation | Use when the business question is about industry attractiveness, competitive position, or durable advantage | Harvard Business Review Press listing: https://store.hbr.org/product/understanding-michael-porter-the-essential-guide-to-competition-and-strategy/12708 |
| Business Model Generation | Business Model Canvas, model design, model differentiation | Use for new ventures, product strategy, GTM, monetization, and operating model decks | Strategyzer: https://www.strategyzer.com/library/business-model-generation |

## Layer 2: Consultant Problem-Solving

These sources help turn messy business input into structured analysis and a recommendation.

| Source | Methods captured | Best use in a deck | Source anchor |
|---|---|---|---|
| The Pyramid Principle, Barbara Minto | Governing thought, top-down logic, grouping, SCQA | Use for answer-first executive communication and action-title slide logic | Barbara Minto: https://www.barbaraminto.com/ |
| The McKinsey Mind | MECE, issue trees, fact-based analysis, synthesis, recommendation communication | Use when the user needs a consulting-style problem-solving deck | Amazon listing: https://www.amazon.com/McKinsey-Mind-Understanding-Implementing-Problem-Solving/dp/0071374299 |
| Bulletproof Problem Solving, Charles Conn and Robert McLean | Define, disaggregate, prioritize, workplan, analyze, synthesize, communicate | Use when the answer is uncertain and the deck must show how the problem was solved | Authors/site: https://bulletproofproblemsolving.com/about-the-authors/ |

These sources help convert a strategic answer into a visual business story.

| Source | Methods captured | Best use in a deck | Source anchor |
|---|---|---|---|
| Playing to Win, A.G. Lafley and Roger Martin | Where to play, how to win, capabilities, management systems | Use for strategy choices, market entry, vertical focus, and tradeoff decks | HBR toolkit: https://hbr.org/store/landing/playing-to-win-strategy-toolkit |

| Duarte / slide:ology / Resonate | Persuasive arc, audience movement, visual hierarchy | Use as support after the consulting POV is clear | Duarte slide:ology: https://www.duarte.com/resources/books/slideology/ |

## Practical Routing Hierarchy

1. If the audience must approve a recommendation, start with Pyramid Principle.
2. If the answer is uncertain, start with McKinsey Mind or Bulletproof Problem Solving.
3. If the user asks for SWOT or a model, use Key Management Models to pick the framework, then turn it into a decision.
4. If the question is about competition, use Porter/Magretta and HBR strategy.
5. If the question is about where to compete and how to win, use Playing to Win.
6. If the question is about a new venture, product strategy, monetization, or GTM model, use Business Model Generation.
7. If the output needs charts, use Say It With Charts and data storytelling as support, not as the core storyline.

## Recommended Buying Order

1. The Pyramid Principle.
2. Key Management Models.
3. HBR's 10 Must Reads on Strategy.
4. The McKinsey Mind or Bulletproof Problem Solving.
5. Business Model Generation.
6. Playing to Win.
7. Say It With Charts.

For the most practical "I need to make strategy PPTs" stack, start with: The Pyramid Principle, Key Management Models, HBR's 10 Must Reads on Strategy, and Say It With Charts.
